JS y preprocesadores

15 de Octubre 2021

JS

Tipos de datos básicos

  • String
  • Number
  • Boolean
  • Array
  • Object
  • Undefined
  • Null

    • Estructuras de datos integradas para JavaScript. Todos los tipos excepto los objetos definen valores inmutables que no se pueden cambiar; estos se nombran como valores primitivos. Boolean representa una entidad lógica, null exactamente un valor, Undefined es algo que no posee valor, etc.

    Preprocesadores CSS

    Es un programa que convierte un tipo de información en hojas de estilo CSS. Los preprocesadores de CSS, extienden las funcionalidades de un CSS común, permitiéndonos tener variables, funciones, mixins, reutilizar código, tener más flexibilidad al momento del desarrollo, etc.

    Dependencias

    node.js permite ejecutar JavaScript en el servidor.

    Como RequireJS, que es una herramienta de gestión de dependencias creada por James Burke, la cual ayuda a manejar los módulos, cargarlos en un orden correcto y combinarlos de forma fácil sin tener que realizar ningún cambio. A su vez, otorga una manera fácil de cargar código una vez cargada la página, permitiendo minimizar el tiempo de descarga.
