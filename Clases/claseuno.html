Como funciona HTML

HTML, siglas en inglés de HyperText Markup Language (‘lenguaje de marcado de hipertexto’), hace referencia al lenguaje de marcado para la elaboración de páginas web. Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros.

  • HTML: Contenido y estructura
  • CSS: Presentación gráfica
  • JS: Comportamiento

    • Arquitecturas

    Para definir la construcción de un sitio, lo primero es investigar para definir quienes serán los usuarios o qué elementos son necesarios para una marca o sitio determinado (Análisis de problemas), luego se definen jerarquías y modos de navegación del sitio web además de objetos y propiedades para definir después la forma en que estas se presentarán visualmente.

    Entonces y en cuanto se tenga claridad de los pasos anteriores se puede avanzar a la creación del Front-end y por consiguiente a la estabilidad, funcionalidad y seguridad del uso del sitio web.

  • Front-end: HTML, CSS, JS
  • Back-end: Sitio web
