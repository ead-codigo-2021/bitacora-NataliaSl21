Introducción a CSS

27 de Agosto 2021

CSS

Cascading Style Sheets (Hojas de estilo de cascada) es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un visitante las visita. Funciona junto al lenguaje HTML, que se encarga del contenido básico de las páginas. Se compone de un selector, una propiedad y un valor.

Se puede relacionar al HTML como archivo externo o bien con la etiqueta style dentro del HTML

Estilos

Los selectores definirán sobre qué elementos se aplicarán las reglas CSS; las cuales pueden ser etiquetas, atributos id (#id) o clases (.class). Además de estas, se pueden encontrar pseudo-clases que corresponden a un estado especial de alguno de los atributos anteriores (:pseudo-clase), del mismo modo que los pseudo-elementos (::pseudo-elemento).

Práctica

Algunos ejemplos de reglas pueden ser el color, la tipografía, el fondo, tamaño de letra, ancho, alto, estilos de párrafo, etc.
