HTML

HyperText Markup Language que básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web, como imágenes, listas, vídeos, etc.

Se componen de una etiqueta de apertura -- Atributo, Valor, Contenido, etc -- y una etiqueta de cierre.

Se ordena mediante anidación; donde una etiqueta podría quedar dentro o al mismo nivel de otra, y estas mismas quedan definidas en indentaciones visuales (distintos niveles).

La estructira básica de un HTML se compone de un HEAD que aparece solo una vez en el documento y declarará información fundamental para máquinas, relaciona hojas de estilo, idioma, etc. y un BODY del cual también solo hay uno por documento y contiene todo aquello que será visible para el usuario en la página web.

Semántica

Refiere a aplicar las etiquetas apropiadas según el contenido, marcar secciones, párrafos, líneas, etc.

  • div: en bloque
  • span: en línea
  • table: tabla
  • img: imagen
  • video: video
  • a: hipervínculos
