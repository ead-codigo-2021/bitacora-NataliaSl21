Introducción a CSS

  • ¿Qué es CSS?

    • CSS son las siglas en inglés para «hojas de estilo en cascada» (cascading style sheets). Básicamente, es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un visitante las visita. Funciona junto al lenguaje HTML, que se encarga del contenido básico de las páginas.

  • Estilos aprendidos en clases

    • Durante clases, vimos ciertos elementos de edición de caracter gráfico, entre ellos el cambio de color de un elemento de texto, familias tipográficas, tamaño y peso de letra y diagramación simple de una página web.

  • Repertorios y fuentes

    • Para la construcción del archivo .css, me dirigí a distintos repertorios de elementos para tomar ideas y tratar de escribirlas en el código. También adjunto páginas que se consultaron en cuanto a la diagramación y orden de párrafos, como centrar textos, centrar imágenes con etiqueta img y colocar márgenes a párrafos.
